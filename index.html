Connect in three steps

Use AntStation Discover for the shortest visual path, or install the CLI buyer, fund it, then pin this peer explicitly.

Step 01

Install and start

Run the local OpenAI-compatible buyer proxy on port 8377.

npm install -g @antseed/cli
antseed buyer start
Step 02

Fund the buyer

Open the payment portal and deposit USDC on Base.

antseed payments
antseed buyer balance
Step 03

Pin this provider

The pin persists across buyer-proxy restarts.

antseed buyer connection set --peer c50de6922b00677c93007c01924586de887ced7b
curl -s http://127.0.0.1:8377/v1/models | jq '.data[].id'

LangChain Python

Use ChatOpenAI with the local AntSeed base URL.

from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="claude-opus-4.8",
    base_url="http://127.0.0.1:8377/v1",
    api_key="antseed",
)
print(llm.invoke("Hello").content)

Hermes agent

Register AntSeed as a custom Chat Completions provider.

model:
  default: claude-opus-4.8
  provider: antseed
custom_providers:
  - name: antseed
    base_url: http://127.0.0.1:8377/v1
    api_key: antseed-p2p
    api_mode: chat_completions
    models:
      - opus-4.7
      - claude-opus-4.8
      - claude-sonnet-5
